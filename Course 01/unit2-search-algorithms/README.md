# Unit 2: Search Algorithms | خوارزميات البحث

### Learning Objectives | أهداف التعلم

By the end of this unit, students will be able to:
- Understand search problems and state spaces
- Implement Breadth-First Search (BFS)
- Implement Depth-First Search (DFS)
- Implement A* search algorithm
- Apply search algorithms to solve pathfinding problems
- Understand heuristic functions

---

## Topics Covered | المواضيع المغطاة

1. **Introduction to Search Problems**
   - State space representation
   - Goal states
   - Search trees

2. **Breadth-First Search (BFS)**
   - Algorithm implementation
   - Queue data structure
   - Completeness and optimality

3. **Depth-First Search (DFS)**
   - Algorithm implementation
   - Stack data structure
   - When to use DFS

4. **A* Search Algorithm**
   - Heuristic functions
   - Cost function (f = g + h)
   - Optimal pathfinding

5. **Applications**
   - Pathfinding in games
   - Puzzle solving
   - Route planning
