# Unit 3: Knowledge Representation | تمثيل المعرفة

### Learning Objectives | أهداف التعلم

By the end of this unit, students will be able to:
- Understand different knowledge representation methods
- Represent knowledge using graphs and trees
- Implement rule-based systems
- Work with knowledge bases
- Build simple expert systems

---

## Topics Covered | المواضيع المغطاة

1. **Knowledge Representation Methods**
   - Logical representation
   - Semantic networks
   - Frames and scripts
   - Production rules

2. **Graph-Based Representation**
   - Knowledge graphs
   - Tree structures
   - Node and edge representation

3. **Rule-Based Systems**
   - IF-THEN rules
   - Forward chaining
   - Backward chaining
   - Rule inference

4. **Knowledge Bases**
   - Storing knowledge
   - Querying knowledge
   - Knowledge updates

5. **Expert Systems Basics**
   - Components of expert systems
   - Simple expert system implementation

---

**Unit Duration:** 2 weeks  
**Difficulty:** Intermediate
