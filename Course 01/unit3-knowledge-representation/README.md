# Unit 3: Knowledge Representation | تمثيل المعرفة

### Learning Objectives | أهداف التعلم

By the end of this unit, students will be able to:
- Understand different knowledge representation methods
- Represent knowledge using graphs and trees
- Implement rule-based systems
- Work with knowledge bases
- Understand expert systems basics

---

## Topics Covered | المواضيع المغطاة

1. **Knowledge Representation Methods**
   - Propositional logic
   - First-order logic basics
   - Semantic networks
   - Frames and scripts

2. **Graph-Based Representation**
   - Knowledge graphs
   - Tree structures
   - Graph traversal for knowledge retrieval

3. **Rule-Based Systems**
   - If-then rules
   - Forward chaining
   - Backward chaining

4. **Knowledge Bases**
   - Facts and rules storage
   - Querying knowledge bases
   - Knowledge inference

5. **Expert Systems**
   - Architecture
   - Inference engine
   - Knowledge acquisition
