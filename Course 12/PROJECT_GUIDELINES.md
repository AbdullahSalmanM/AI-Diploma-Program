# Capstone Project Guidelines | إرشادات مشروع التخرج
## AIAT 126

---

## Project Requirements | متطلبات المشروع

### Scope
- **Duration:** 8-12 weeks
- **Complexity:** Must demonstrate mastery of multiple AI concepts
- **Deliverables:** Working system + documentation + presentation

### Evaluation Criteria
1. Problem complexity and relevance (20%)
2. Technical implementation (30%)
3. Code quality and documentation (20%)
4. Deployment and usability (15%)
5. Presentation and communication (15%)

---

## Project Phases | مراحل المشروع

### Phase 1: Proposal (Weeks 1-2)
- Problem identification
- Requirements gathering
- Technical approach
- Timeline planning

### Phase 2: Development (Weeks 3-8)
- Data collection/preparation
- Model development
- System integration
- Testing

### Phase 3: Deployment (Weeks 9-10)
- Production deployment
- Monitoring setup
- Documentation

### Phase 4: Presentation (Weeks 11-12)
- Final report
- Demo preparation
- Presentation
