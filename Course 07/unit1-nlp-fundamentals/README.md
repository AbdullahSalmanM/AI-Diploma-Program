# Unit 1: NLP Fundamentals | أساسيات معالجة اللغة الطبيعية

### Learning Objectives | أهداف التعلم

By the end of this unit, students will be able to:
- Understand what Natural Language Processing is
- Perform text preprocessing
- Implement tokenization
- Understand word embeddings basics
- Work with NLTK and spaCy

---

## Topics Covered | المواضيع المغطاة

1. **Introduction to NLP**
   - What is NLP?
   - NLP applications
   - Challenges in NLP

2. **Text Preprocessing**
   - Lowercasing
   - Removing punctuation
   - Stop word removal
   - Stemming and lemmatization

3. **Tokenization**
   - Word tokenization
   - Sentence tokenization
   - Subword tokenization

4. **Basic NLP Libraries**
   - NLTK basics
   - spaCy basics
   - TextBlob basics
