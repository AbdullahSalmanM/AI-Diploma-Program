## Technical Details

- **Extraction Method**: Python script using `python-pptx` and `pdfplumber`/`PyPDF2`
- **Format**: Markdown with organized sections by material/file
- **Encoding**: UTF-8 (supports Arabic/English content)
- **Structure**: Organized by material type (PPTX Files, PDFs) and file name

## Usage

Students can:
1. Open `COURSE_SUMMARY.md` in any text editor
2. Search for specific topics using Ctrl+F (Cmd+F on Mac)
3. Read through all course content without opening original files
4. Use as a study guide or quick reference
