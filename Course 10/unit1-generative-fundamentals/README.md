# Unit 1: Generative Models Fundamentals | أساسيات النماذج التوليدية

### Learning Objectives | أهداف التعلم

By the end of this unit, students will be able to:
- Understand generative vs discriminative models
- Understand probability distributions in generative models
- Work with latent spaces
- Understand basic generative model architectures
- Generate simple synthetic data

---

## Topics Covered | المواضيع المغطاة

1. **Generative vs Discriminative Models**
   - Key differences
   - When to use each
   - Examples of each type

2. **Probability in Generative Models**
   - Joint probability
   - Conditional probability
   - Prior and posterior distributions

3. **Latent Spaces**
   - What is a latent space?
   - Encoding and decoding
   - Sampling from latent space

4. **Basic Generative Models**
   - Naive Bayes (generative classifier)
   - Gaussian Mixture Models
   - Introduction to VAEs and GANs

---

**Unit Duration:** 2 weeks  
**Difficulty:** Advanced  
**Prerequisites:** Completion of Courses 01-09, strong understanding of probability and deep learning
