# Unit 1: Generative Models Fundamentals | أساسيات النماذج التوليدية

### Learning Objectives | أهداف التعلم

By the end of this unit, students will be able to:
- Understand generative vs discriminative models
- Work with latent spaces
- Understand probability distributions in generation
- Implement basic generative models

---

## Topics Covered | المواضيع المغطاة

1. **Generative vs Discriminative**
   - Differences
   - When to use each
   - Examples

2. **Latent Space**
   - Concept of latent variables
   - Encoding and decoding
   - Dimensionality

3. **Probability in Generation**
   - Probability distributions
   - Sampling
   - Likelihood

4. **Basic Generative Models**
   - Simple autoencoders
   - Basic generation techniques
